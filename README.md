# 🚀 JobNest – Job Seeking Portal

JobNest is a modern full-stack job portal that connects job seekers with recruiters. It provides a seamless platform to search, apply, and manage job opportunities with an intuitive and responsive UI.

---

## ✨ Features

### 👤 Authentication

* Secure Login & Signup
* Role-based access (Job Seeker / Recruiter)

### 🔍 Job Seeker

* Browse and search jobs
* Filter jobs by title/location
* View detailed job descriptions
* Apply for jobs easily
* Track application status

### 🧑‍💼 Recruiter

* Post new job listings
* Manage posted jobs
* View applicants
* Dashboard with job stats

### 🎨 UI/UX

* Fully responsive design (mobile + desktop)
* Clean and modern interface
* Smooth user experience

## ⚙️ Installation & Setup


```bash

```

